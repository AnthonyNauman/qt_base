# Qt5 Desktop Application Template

A template project for creating cross-platform desktop applications using Qt5 with Docker-based builds for Linux and Windows.

## Features

- 🚀 Ready-to-use Qt5 project structure
- 🐳 Docker builds for Linux (x86_64) and Windows (x86_64)
- 📦 Automatic dependency installation inside containers
- 🔧 CMake support
- 🎨 Pre-configured code style (clang-format)

## Requirements

- Docker
- Docker Compose (optional)
- CMake (for local builds)
- Qt5 (for local builds)
